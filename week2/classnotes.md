```
<script defer></script> 
```
-> The 'defer' keyword makes the script runs later until most of the page are initialized.
ex. While using FB/ IG, the page loads more contents as you scroll to the bottom.
    The defer keyword will make the js code run after the page loads then it triggers the js code.
    However, there are more content(html) coming up at the bottom, but the js code is already applied to the page 'first'
